## 예시 답변
질문: 요한복음의 내용을 요약해 
답변: 요한복음은 예수 그리스도의 신성과 그의 사역을 중심으로 한 복음서입니다. 이 복음서는 다른 세 복음서와는 달리 예수님의 신적 본성과 그의 가르침에  더 중점을 두고 있습니다. 주요 내용은 다음과 같습니다:

1. **예수님의 신성**: 요한복음은 예수님을 "말씀" (로고스)으로 소개하며, 그가 하나님과 함께 있었고 하나님이라는 것을 강조합니다 (요한복음 1:1-14).     

2. **기적과 표적**: 예수님은 여러 기적을 행하시며, 이를 통해 그의 신성을 드러내십니다. 물을 포도주로 변화시키는 사건, 병자를 고치는 사건, 오병이어의  기적 등이 포함됩니다 (요한복음 2-6장).

3. **가르침**: 예수님은 여러 비유와 교훈을 통해 하나님 나라에 대해 가르치십니다. "나는 생명의 떡이다", "나는 세상의 빛이다", "나는 선한 목자다"와 같은 말씀을 통해 자신의 정체성을 드러냅니다 (요한복음 6:35, 8:12, 10:11).

4. **십자가의 죽음과 부활**: 요한복음은 예수님의 십자가에서의 죽음과 부활을 중심으로 전개됩니다. 예수님은 인류의 죄를 대신하여 죽으시고, 부활하심으로 써 영원한 생명을 약속하십니다 (요한복음 19-20장).

5. **성령의 약속**: 예수님은 제자들에게 성령을 보내겠다고 약속하시며, 성령이 그들을 인도하고 가르칠 것이라고 말씀하십니다 (요한복음 14:16-17).
